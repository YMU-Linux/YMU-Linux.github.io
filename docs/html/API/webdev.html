Web Protocols

  • HTTPS: HTTP over TLS/SSL for secure communication.

  • FTP (File Transfer Protocol): For transferring files between

client and server. - SFTP (SSH File Transfer Protocol): Secure alternative to FTP ove r SSH. - WebSocket (WS/WSS): Full-duplex communication between client and server for real-time applications. - SPDY: Predecessor to HTTP/2, developed by Google to improve web performance. - SMTP (Simple Mail Transfer Protocol): Often used in web apps for sending emails. - IMAP/POP3: Protocols for retrieving emails in webmail systems. - DNS (Domain Name System): Resolves domain names to IP addresses. - TLS/SSL: Underlying cryptographic protocols for HTTPS and secure communications.
